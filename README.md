# Steam Minimal C++

## About

A port of the Python SteamClient to be C++.
This will not include most of the logic, just enough for my remake of
ACCELA-min to C++ with GUI.

## Requirements

- libcurl
- protobuf

## Building

### Step 1: Cross Platform

1. Clone the repo
```bash
git clone https://git.petar.cc/gee.wzz/steam-min-cpp.git
```

2. CD into the repo
```bash
cd steam-min-cpp
```

3. Make a build folder
```bash
mkdir build
```

4. CD into the build folder
```bash
cd build
```
